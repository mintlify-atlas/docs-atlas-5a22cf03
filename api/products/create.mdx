---
title: "Create Product"
api: "POST /products"
---

Create a new product in the system.

## Authentication

This endpoint requires authentication with the `admin` role.

## Request Body

<ParamField body="name" type="string" required>
  The product name (max 60 characters)
</ParamField>

<ParamField body="price" type="number" required>
  The product price (max 2 decimal places)
</ParamField>

<ParamField body="image" type="string" required>
  The product image URL or filename
</ParamField>

<ParamField body="inventory" type="number" required>
  The quantity in stock (integer)
</ParamField>

<ParamField body="categoryId" type="integer" required>
  The ID of the category this product belongs to
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The unique identifier for the product
</ResponseField>

<ResponseField name="name" type="string">
  The product name
</ResponseField>

<ResponseField name="price" type="number">
  The product price
</ResponseField>

<ResponseField name="image" type="string">
  The product image URL or filename
</ResponseField>

<ResponseField name="inventory" type="number">
  The quantity in stock
</ResponseField>

<ResponseField name="categoryId" type="integer">
  The category ID
</ResponseField>

<ResponseField name="category" type="object">
  The category object with full details
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.example.com/products \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Espresso Coffee",
    "price": 12.99,
    "image": "espresso.jpg",
    "inventory": 50,
    "categoryId": 1
  }'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/products', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Espresso Coffee',
    price: 12.99,
    image: 'espresso.jpg',
    inventory: 50,
    categoryId: 1
  })
});

const product = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.example.com/products',
    headers={'Authorization': 'Bearer YOUR_TOKEN'},
    json={
        'name': 'Espresso Coffee',
        'price': 12.99,
        'image': 'espresso.jpg',
        'inventory': 50,
        'categoryId': 1
    }
)

product = response.json()
```
